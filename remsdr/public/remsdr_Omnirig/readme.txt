This directory is dedicated to interface Omnirig with Remote SDR.
The file RemoteSDR.ini is to be copied in your OmiRig Rigs description directory on your PC. Generally C://Programmes(x86)/Afreet/Omnirig/Rigs.
The file RemoteSDR.vspe is to be copied on your PC. It defines the "Virtual Serial Port Emulator" which interconnect Omnirig on a virtual serial port on one side, and your Remote SDR Single Board processor via the LAN on the other side.
